# Landing Page — Margarit Construcciones

Proyecto fuente de la landing page para **Margarit Construcciones**, empresa chilena de construcción y remodelación.

## Tecnologías

- **React** + **TypeScript**
- **Vite** (bundler)
- CSS personalizado (`index.css`)

## Estructura del proyecto

| Archivo / Carpeta | Descripción |
|---|---|
| `App.tsx` | Componente raíz de la aplicación |
| `Home.tsx` | Página principal de la landing page |
| `TestimonialsCarousel.tsx` | Componente de carrusel de testimonios |
| `index.css` | Estilos globales |
| `index.html` | Punto de entrada HTML |
| `ideas.md` | Notas e ideas del proyecto |
| `assets/` | Imágenes y recursos gráficos |

## Desarrollo local

```bash
npm install
npm run dev
```

## Despliegue

El proyecto está configurado para desplegarse en **GitHub Pages** mediante GitHub Actions.
